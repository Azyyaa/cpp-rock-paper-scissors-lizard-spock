# Rock Paper Scissors

A console-based Rock Paper Scissors game written in C++.

## Features

* Main menu system
* New Game
* Rules
* Settings
* Exit

### Game Modes

* Easy Mode: Rock, Paper, Scissors
* Hard Mode: Rock, Paper, Scissors, Lizard, Spock

### Customization

* Change background color
* Change text color

### Gameplay

* Player vs Computer
* Randomized computer choices
* Win, lose, and draw detection

## Technologies

* C++
* Functions
* Conditional statements
* Loops
* Menus and user input handling

## Goal

The goal of this project was to practice menu-driven applications, game logic, user settings, and customization features in C++.
